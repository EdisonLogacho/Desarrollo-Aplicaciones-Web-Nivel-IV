Que significa HTML (HyperText Markup Language)

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web y sirve para estructurar el contenido que se muestra en el navegador mediante etiquetas, organizando elementos como texto, imágenes, videos y enlaces, además de permitir la integración de otros recursos como CSS para diseño y JavaScript para de permitir la integración de otros recursos como CSS para diseño y JavaScript para interactividad; en resumen, es la base fundamental de cualquier sitio web.

Etiqueta Significado Características
"H1" Título principal de la página - Solo debe haber uno por página. - Es el más importante para SEO. - Tamaño más grande por defecto.
"H2" Subtítulo principal - Divide secciones dentro del contenido. - Puede haber varios. - Segundo nivel de importancia.
"H3" Subtítulo secundario - Detalla subsecciones de. - Tamaño menor que.
